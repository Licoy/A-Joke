### :jack_o_lantern: { 第1档 } 国产的东西好用

### :jack_o_lantern: { 第4档 } 觉得我帅的请亮灯
我们楼层都是声控灯，只要声音较大，灯都会亮。有一天我和我的朋友在楼底下玩，突然朋友对着楼栋大喊：觉得我帅的请亮灯！整栋楼的灯都亮了。我靠。我只想对他说：女嘉宾们都没睡醒。。。。
